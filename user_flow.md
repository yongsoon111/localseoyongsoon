# User Flow & UX Specification
## Local SEO 순위 분석 보고서 생성 도구

---

## 1. 사용자 페르소나

### Primary User: 마케팅 에이전시 / SEO 전문가
- **목적**: 고객에게 Local SEO 순위 분석 보고서 제공
- **니즈**:
  - 빠르고 정확한 순위 분석
  - 전문적인 보고서 자동 생성
  - 고객 전송 자동화 (이메일/SMS/웹훅)
- **기술 수준**: 중급 (Google Maps URL 복사 가능)
- **사용 빈도**: 주 5-10회

### Secondary User: 비즈니스 오너
- **목적**: 자신의 매장 순위 확인
- **니즈**: 간단한 분석, 경쟁사와 비교
- **기술 수준**: 초급
- **사용 빈도**: 월 1-2회

## 5. 자동화 워크플로우

### 5.1 자동 보고서 생성 플로우
```
스캔 완료
    ↓
데이터 수집 완료 확인
    ↓
보고서 템플릿 선택
    ↓
고객 정보 자동 입력 (DB에서)
    ↓
히트맵 이미지 생성
    ↓
차트 이미지 생성
    ↓
인사이트 AI 생성 (선택)
    ↓
PDF 렌더링 (WeasyPrint 또는 Puppeteer)
    ↓
파일 저장 (S3 또는 로컬)
    ↓
전송 대기열에 추가
```

### 5.2 전송 자동화 플로우
```
보고서 생성 완료
    ↓
전송 방법 확인 (이메일/SMS/웹훅)
    ↓
┌─────────┬─────────┬─────────┐
│ 이메일   │  SMS    │ 웹훅    │
├─────────┼─────────┼─────────┤
│ SMTP    │ Twilio  │ HTTP    │
│ SendGrid│ API     │ POST    │
└─────────┴─────────┴─────────┘
    ↓
전송 시도 (최대 3회 재시도)
    ↓
전송 성공 로그 기록
    ↓
사용자에게 알림 (대시보드 + 이메일)
```
